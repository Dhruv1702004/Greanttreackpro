# GrantTrack Pro
A smart grant management system for schools.

## How to Run
1. Run `npm install`
2. Run `node server.js`
3. Access at http://localhost:3000

## Live Deployment
Use Node/Express hosting, not static hosting.

- Build Command: `npm install`
- Start Command: `npm start`
- Main file: `server.js`
- The server uses `process.env.PORT || 3000`, so it works locally and on live hosting platforms.
